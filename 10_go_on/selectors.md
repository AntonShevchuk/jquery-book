–ö–∞–∫ —è —É–∂–µ –≥–æ–≤–æ—Ä–∏–ª —Ä–∞–Ω–µ–µ, –≤ –ø–æ–∏—Å–∫–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –Ω–∞ —Å—Ç—Ä–∞–Ω–∏—Ü–µ –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –ø—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏ –ø–æ–ª–æ–≤–∏–Ω–∞ —É—Å–ø–µ—à–Ω–æ–π —Ä–∞–±–æ—Ç—ã —Å jQuery.
–¢–∞–∫ —á—Ç–æ –ø—Ä–∏—Å—Ç—É–ø–∏–º –∫ –ø–æ–∏—Å–∫–∞–º –ø–æ –¥–æ–∫—É–º–µ–Ω—Ç—É (—á—Ç–æ–±—ã –Ω–µ –ª–∏—Å—Ç–∞—Ç—å, –ø—É—Å—Ç—å –ø—Ä–∏–º–µ—Ä HTML –±—É–¥–µ—Ç –∏ —Ç—É—Ç):

–ê —Ç–µ–ø–µ—Ä—å –ø—Ä–∏—Å—Ç—É–ø–∏–º –∫ –≤—ã–±–æ—Ä–∫–∞–º ‚Äî –≤—ã–±–æ—Ä —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –ø–æ ¬´id¬ª –ª–∏–±–æ –∏–º–µ–Ω–∏ –∫–ª–∞—Å—Å–∞, –∞–Ω–∞–ª–æ–≥–∏—á–Ω–æ –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–º –≤ CSS:

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("#content")</a> ‚Äì –≤—ã–±–∏—Ä–∞–µ–º —ç–ª–µ–º–µ–Ω—Ç —Å ¬´id="content"¬ª

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("section#content")</a> ‚Äì –≤—ã–±–∏—Ä–∞–µ–º `<section>` —Å ¬´id="content"¬ª (—Ö–æ—Ç—è –∏ –±–µ–∑ ¬´section —Ä–∞–±–æ—Ç–∞–µ—Ç)

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$(".intro")</a> ‚Äì –≤—ã–±–∏—Ä–∞–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã —Å ¬´class="intro"¬ª

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("p.intro")</a> ‚Äì –≤—ã–±–∏—Ä–∞–µ–º –≤—Å—ë ¬´p¬ª —Å ¬´class="intro"¬ª

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$(".intro.pinned")</a> ‚Äì –≤—ã–±–∏—Ä–∞–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã —Å –∫–ª–∞—Å—Å–∞–º–∏ ¬´intro¬ª –∏ ¬´pinned¬ª

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("h3")</a> ‚Äì –≤—ã–±–∏—Ä–∞–µ–º –≤—Å–µ —Ç–µ–≥–∏ `<h3>`

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("h1, h2")</a> ‚Äì –≤—ã–±–∏—Ä–∞–µ–º –≤—Å–µ —Ç–µ–≥–∏ `<h1>` –∏ `<h2>`

_–ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ –≤–∞–ª–∏–¥–Ω—ã–µ –∏–º–µ–Ω–∞ –∫–ª–∞—Å—Å–æ–≤ –∏ –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä–æ–≤_

–¢–µ–ø–µ—Ä—å –≤—Å–ø–æ–º–Ω–∏–º, —á—Ç–æ –º—ã –≤ DOM–µ –Ω–µ –æ–¥–Ω–∏, —ç—Ç–æ-—Ç–∞–∫–∏ –∏–µ—Ä–∞—Ä—Ö–∏—á–µ—Å–∫–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞:

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("article h3")</a> ‚Äì –≤—ã–±–∏—Ä–∞–µ–º –≤—Å–µ —Ç–µ–≥–∏ `<h2>` –≤–Ω—É—Ç—Ä–∏ —Ç–µ–≥–∞ `<article>`

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("article").find("h3")</a> ‚Äì –∞–Ω–∞–ª–æ–≥–∏—á–Ω–æ –ø—Ä–∏–º–µ—Ä—É –≤—ã—à–µ

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("section article h3")</a> ‚Äì –≤—ã–±–∏—Ä–∞–µ–º –≤—Å–µ —Ç–µ–≥–∏ `<h3>` –≤–Ω—É—Ç—Ä–∏ —Ç–µ–≥–∞ `<article>`, –∫–æ—Ç–æ—Ä—ã–µ –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤–Ω—É—Ç—Ä–∏ —Ç–µ–≥–∞ `<section>`, _–≤ DOM–µ –∫–æ—Ç–æ—Ä—ã–π –ø–æ—Å—Ç—Ä–æ–∏–ª –î–∂–µ–∫_
  
<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("section").find("article").find("h3")</a> ‚Äì –∏ –µ—â—ë —Ä–∞–∑, –Ω–æ –Ω–∞ –¥—Ä—É–≥–æ–π –ª–∞–¥

–£ –Ω–∞—Å –µ—Å—Ç—å —Å–æ—Å–µ–¥–∏, –∏ —É –Ω–∞—Å —Å –Ω–∏–º–∏ –Ω–∞–ª–∞–∂–µ–Ω –∫–æ–Ω—Ç–∞–∫—Ç. –í–æ—Ç –≤–∞–º –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å–ø–æ—Å–æ–±–æ–≤ –∫–∞–∫ –Ω–∞–π—Ç–∏ –æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ —ç–ª–µ–º–µ–Ω—Ç:

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("article + article")</a> ‚Äì –≤—ã–±–æ—Ä –≤—Å–µ—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ `<article>`, –ø–µ—Ä–µ–¥ –∫–æ—Ç–æ—Ä—ã–º–∏ –µ—Å—Ç—å —Ç–µ–≥ `<article>`

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("#stick ~ article")</a> ‚Äì –≤—ã–±–æ—Ä –≤—Å–µ—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ `<article>` –ø–æ—Å–ª–µ —ç–ª–µ–º–µ–Ω—Ç–∞ —Å ¬´id="stick"¬ª

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("#stick").next()</a> ‚Äì –≤—ã–±–æ—Ä —Å–ª–µ–¥—É—é—â–µ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ –ø–æ—Å–ª–µ —ç–ª–µ–º–µ–Ω—Ç–∞ —Å ¬´id="stick"¬ª

–†–æ–¥—Å—Ç–≤–µ–Ω–Ω—ã–µ —Å–≤—è–∑–∏:

`$("*")` ‚Äì –≤—ã–±–æ—Ä –≤—Å–µ—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤; **–Ω–∏–∫–æ–≥–¥–∞ –Ω–µ –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ!**

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("article > h3")</a> ‚Äì –≤—ã–±–∏—Ä–∞–µ–º –≤—Å–µ —Ç–µ–≥–∏ `<h3>`, –∫–æ—Ç–æ—Ä—ã–µ —è–≤–ª—è—é—Ç—Å—è –Ω–µ–ø–æ—Å—Ä–µ–¥—Å—Ç–≤–µ–Ω–Ω—ã–º–∏ –ø–æ—Ç–æ–º–∫–∞–º–∏ —Ç–µ–≥–∞ `<article>`

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("article > *")</a> ‚Äì –≤—ã–±–æ—Ä –≤—Å–µ—Ö –ø–æ—Ç–æ–º–∫–æ–≤ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ `<article>`

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("article").children()</a> ‚Äì –∞–Ω–∞–ª–æ–≥–∏—á–Ω–æ –ø—Ä–∏–º–µ—Ä—É –≤—ã—à–µ

<a class="jqbook" href="#" data-target="#html-example" data-type="highlight">$("p").parent()</a> ‚Äì –≤—ã–±–æ—Ä –≤—Å–µ—Ö –ø—Ä—è–º—ã—Ö –ø—Ä–µ–¥–∫–æ–≤ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ `<p>`

_–ï—Å–ª–∏ —Ö–æ—Ç–∏—Ç–µ –ø–æ–∏–≥—Ä–∞—Ç—å—Å—è —Å —Å–µ–ª–µ–∫—Ç–æ—Ä–∞–º–∏ –æ—Ç –¥—É—à–∏, —Ç–æ –æ—Ç–∫—Ä–æ–π—Ç–µ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â—É—é —Å—Ç—Ä–∞–Ω–∏—á–∫—É [css.selectors.html](../code/css.selectors.html) –≤ –Ω–æ–≤–æ–π –≤–∫–ª–∞–¥–∫–µ, –∏ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –∫–æ–Ω—Å–æ–ª–∏ –∑–∞–ø—É—Å—Ç–∏—Ç–µ —Å–∫—Ä–∏–ø—Ç `$("p").effect("highlight")`_

### –ü–æ–∏—Å–∫ –ø–æ –∞—Ç—Ä–∏–±—É—Ç–∞–º

–ï—â—ë —Å–æ –≤—Ä–µ–º—ë–Ω CSS 2 –±—ã–ª–∞ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –Ω–∞–π—Ç–∏ —ç–ª–µ–º–µ–Ω—Ç —Å –æ–ø—Ä–µ–¥–µ–ª—ë–Ω–Ω—ã–º–∏ –∞—Ç—Ä–∏–±—É—Ç–∞–º–∏, –≤ CSS 3 –¥–æ–±–∞–≤–∏–ª–∏ –µ—â—ë –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–µ–π –ø–æ –ø–æ–∏—Å–∫—É:

`a[href]` ‚Äî –≤—Å–µ —Å—Å—ã–ª–∫–∏ —Å –∞—Ç—Ä–∏–±—É—Ç–æ–º ¬´href¬ª

`a[href=#]` ‚Äî –≤—Å–µ —Å—Å—ã–ª–∫–∏ —Å ¬´href=#¬ª

`a[href~=#]` ‚Äî –≤—Å–µ —Å—Å—ã–ª–∫–∏ —Å ¬´#¬ª –≥–¥–µ-—Ç–æ –≤ ¬´href¬ª

`a[hreflang|=en]` ‚Äî –≤—Å–µ —Å—Å—ã–ª–∫–∏, –¥–ª—è –∫–æ—Ç–æ—Ä—ã—Ö hreflang –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å ¬´en¬ª –∏ –æ–±—Ä–µ–∑–∞–µ—Ç—Å—è –ø–æ —Å–∏–º–≤–æ–ª—É ¬´-¬ª ‚Äî ¬´en¬ª, ¬´en-US¬ª, ¬´en-UK¬ª

`a[href^=http]` ‚Äî —Å—Å—ã–ª–∫–∏, –Ω–∞—á–∏–Ω–∞—é—â–∏–µ—Å—è —Å ¬´http¬ª

`a[href*="google.com"]` ‚Äî —Å—Å—ã–ª–∫–∏ –Ω–∞ –ø–æ–≥—É–≥–ª–∏—Ç—å

`a[href$=.pdf]` ‚Äî —Å—Å—ã–ª–∫–∏ –Ω–∞ PDF-—Ñ–∞–π–ª—ã (–ø–æ –∏–¥–µ–µ)

_–ó–∞–≥–ª—è–Ω—É–≤ –≤–Ω—É—Ç—Ä—å jQuery, –≤—ã, —Å–∫–æ—Ä–µ–π –≤—Å–µ–≥–æ, –Ω–∞–π–¥—ë—Ç–µ —Ç–æ –º–µ—Å—Ç–æ, –≥–¥–µ –≤–∞—à–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –±—É–¥–µ—Ç –∞–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞—Ç—å—Å—è —Å –ø–æ–º–æ—â—å—é —Ä–µ–≥—É–ª—è—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏–π, –ø–æ —ç—Ç–æ–π –ø—Ä–∏—á–∏–Ω–µ –≤ —Å–µ–ª–µ–∫—Ç–æ—Ä–∞—Ö –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞—Ç—å —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–µ —Å–∏–º–≤–æ–ª—ã, –∏—Å–ø–æ–ª—å–∑—É—è –¥–≤–æ–π–Ω–æ–π –æ–±—Ä–∞—Ç–Ω—ã–π —Å–ª–µ—à ¬´\\¬ª:_

```javascript
$("a[href^=\\/]").addClass("internal");
```

### –ü–æ–∏—Å–∫ –ø–æ –¥–æ—á–µ—Ä–Ω–∏–º —ç–ª–µ–º–µ–Ω—Ç–∞–º

–•–æ—Ç–µ–ª–æ—Å—å –±—ã –µ—â—ë –æ–±—Ä–∞—Ç–∏—Ç—å –≤–Ω–∏–º–∞–Ω–∏–µ –Ω–∞ —Å–µ–ª–µ–∫—Ç–æ—Ä—ã –∏–∑ —Å–ø–µ—Ü–∏—Ñ–∏–∫–∞—Ü–∏–∏ [CSS 3](http://www.w3.org/TR/css3-selectors/) ‚Äî –º–Ω–æ–≥–æ –∏–Ω—Ç–µ—Ä–µ—Å–Ω—ã—Ö:
